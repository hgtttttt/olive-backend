# olive-backend
橄榄支项目后端1.0版
为节省开发时间，本项目由软工项目已有的代码改写而来，因此代码中可能会出现一些废弃无用部分、未启用部分。现对项目结构说明如下：

* blog：论坛
* collected_static：废弃
* comment：评论部分，尚未修改适应本项目
* course：课程部分
* media：原存储图片使用，废弃
* message：私人消息部分，尚未修改适应本项目
* report：报告部分，尚未修改适应本项目
* saadback：项目配置
* search：废弃
* team：支教队部分
* user：用户部分
* tool：一些辅助功能

当前数据库部署在rcy个人服务器的mysql上，后续需要迁移到团队服务器。
